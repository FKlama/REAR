REAR is a small project I work on for the University of GÃ¶ttingen to use in spoken language exams as the recording device.

Features

Client:

  • Records Mono Audio into FLAC
  • Client Software ignores user actions
    • Can't close window
    • Can't interrupt recording
  • Transfer recorded file to server over a SSH tunnel

Controller:

  • Allows controlling multiple clients from a central machine
  • Overview over client status
  • Grouping of clients
  • Ability to save and load groups and client lists

Server:

  • Transcode FLAC => MP3
